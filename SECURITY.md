# Security Policy

## Supported Versions

We currently support the latest released version.

## Reporting a Vulnerability

Please **do not** file public issues for security-sensitive reports.

- Email: zac.moltbot@gmail.com
- Subject: `[lobster-room] security report`

Include:
- A description of the issue and impact
- Reproduction steps / PoC if available
- Affected version(s)

We will acknowledge receipt and coordinate a fix/release.
