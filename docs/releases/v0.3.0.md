# Lobster Room v0.3.0

Released: 2026-04-09

## Highlights

- Bundled starter rooms with baked-in backgrounds and manual walk maps
- Canonical agent identity fixes so feed, Now panel, and room residents agree on who is acting
- Room switching / reset / rehydrate consistency fixes
- Cleaner Settings layout grouped by Room Setup / Appearance / History & Agents
- Feed UX polish: better task wording, stronger parent/child attribution, fewer duplicate or `@unknown` rows
- Refreshed homepage demo GIF and README copy

## Upgrade notes

- Restart the OpenClaw Gateway after updating so new plugin assets are served
- Prefer pinned release installs (`VERSION=v0.3.0`) over branch-tip installs
- Existing custom rooms remain supported; bundled rooms are additional starter content
